# Amazonia

Portfolio / catálogo de casas en seco construido con Next.js (App Router) + TypeScript + TailwindCSS.

## Requisitos
- Node.js 18+
- npm

## Configuración
```bash
npm install
npm run dev
```

## Rutas
- `/` catálogo con filtros y búsqueda
- `/casas/[slug]` detalle de cada casa
- `/como-construimos` proceso constructivo
- `/contacto` formulario y datos de contacto
